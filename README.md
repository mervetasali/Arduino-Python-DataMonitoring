Arduino-Python-DataMonitoring

A hands-on project series focused on learning serial communication between Arduino and Python.

Project Goal

The purpose of this project is to build a solid foundation for:

Serial communication
Data monitoring
Data validation
Logging
Real-time dashboards
SCADA systems
Industrial automation concepts
Current Project
Arduino System Monitor

Arduino generates random temperature values and sends them to Python through serial communication.

Example output:

TEMP:25
TEMP:28
TEMP:21
TEMP:30
Current Version
v0.0.0

Features:

Arduino temperature simulator
Random temperature generation
Serial communication at 9600 baud
Data transmission every second
